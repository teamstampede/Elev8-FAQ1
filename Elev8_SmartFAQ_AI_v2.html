🔧 Google Sheets Live Sync — Setup Guide

Follow these steps once to connect this FAQ tool to your Google Sheet. After setup, any question added by any staff member on any campus will appear for everyone automatically. This only needs to be done one time by whoever sets up the system.

Quick Config — Paste Your Sheet ID Here

Once you have your Google Sheet set up (Step 2 below), paste your Sheet ID here and click Save. The Sheet ID is the long string in your Sheet URL between /d/ and /edit.

✅ Sheet ID saved! Click "Sync Now" in the toolbar to load questions.
1

Create a Google Sheet

Go to sheets.google.com and create a new blank spreadsheet. Name it "Elev8 FAQ Database". In Row 1, add these exact column headers in order:

id | dept | cat | tags | q | family | staff

Note: You can copy and paste the built-in FAQ data from the Export CSV button on the Manage tab to pre-populate the Sheet.

2

Publish the Sheet as a Web App (Read Access)

This is what allows the FAQ tool to read your Sheet without requiring staff to log in. In your Google Sheet:

File → Share → Publish to web → Sheet1 → CSV → Publish

Copy the published URL. Your Sheet ID is the long string in your Sheet's regular URL between /d/ and /edit. Paste just the Sheet ID in the Quick Config box above.

3

Set Up Write Access (for Adding Questions)

To allow staff to add questions from the FAQ tool, you need a Google Apps Script Web App. Here is the script — paste it into your Sheet's Script Editor (Extensions → Apps Script):

function doPost(e) { var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); var data = JSON.parse(e.postData.contents); sheet.appendRow([ data.id, data.dept, data.cat, data.tags, data.q, data.family, data.staff ]); return ContentService.createTextOutput(JSON.stringify({status:'ok'})) .setMimeType(ContentService.MimeType.JSON); } function doDelete(e) { var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); var id = e.parameter.id; var data = sheet.getDataRange().getValues(); for (var i = 1; i < data.length; i++) { if (data[i][0] === id) { sheet.deleteRow(i + 1); break; } } return ContentService.createTextOutput(JSON.stringify({status:'ok'})) .setMimeType(ContentService.MimeType.JSON); }

After pasting the script: click Deploy → New deployment → Web App. Set Execute as: Me and Who has access: Anyone. Copy the Web App URL and paste it below:

✅ Web App URL saved! The Add Question tab will now sync to Google Sheets.
4

Host on GitHub Pages (Access from Any Campus)

To make this tool available from every campus at a single URL:

1. Go to github.com and create a free account

2. Create a new repository named elev8-faq — set it to Private

3. Upload this HTML file to the repository

4. Go to Settings → Pages → Source: main branch → Save

5. Your FAQ tool will be live at: https://[your-username].github.io/elev8-faq

Share this URL only with authorized staff. Since the repository is private, the tool content is not indexed or discoverable by search engines.

5

Control Staff Access

To view the FAQ tool: Share the GitHub Pages URL only with authorized staff. No login required — URL access is the control.

To add/edit questions via the Sheet directly: In Google Sheets, click Share and add only the staff email addresses you authorize. They can add rows directly in the Sheet — changes appear in the tool on next sync.

To revoke access: Remove their email from the Sheet's sharing settings. Their ability to edit is immediately revoked.
